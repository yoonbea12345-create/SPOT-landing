# sajutight 1% 랜딩페이지 분석

## 디자인 특징
- **배경**: 흑백 고화질 이미지 (체스말 쥔 손) — 강렬한 시각적 임팩트
- **레이아웃**: 모바일 세로형 풀스크린 1페이지 구조
- **색상**: 흑백 배경 + 빨간색 포인트 텍스트 ("1%의 전략")
- **폰트**: 굵은 명조/세리프 계열 — 고급스럽고 무게감 있음
- **하단 CTA**: 고정 하단바 — "내 100년 종합사주 보러가기" (어두운 배경 + 점선 테두리)

## 문구 구조 (핵심)
1. 문제 제기 (회색, 작은 글씨): "당신이 지금까지 잘 안 풀린 것은.."
2. 반박 (회색, 중간): "못해서도 / 노력이 부족해서도 아닌"
3. 핵심 메시지 (빨간 강조 + 흰색, 큰 글씨): "1%의 전략을 / 몰라서였습니다"
4. CTA (하단 고정): "내 100년 종합사주 보러가기"

## 구조 패턴 (SPOT에 적용 가능)
- 문제 제기 → 기존 방식 부정 → 핵심 솔루션 강조 → CTA
- 텍스트가 이미지 위에 오버레이되어 강렬한 인상
- 문구가 극도로 짧고 리듬감 있음 (3~5단어 단위)
- 단 하나의 CTA만 존재 (분산 없음)
- 스크롤 없이 1화면에 모든 핵심 메시지 전달
