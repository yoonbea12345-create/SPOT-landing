# SS.pdf 핵심 내용 정리

## 핵심 슬로건/문구
- "이젠 과거로부터의 선택에서 실시간의 탐색을."
- "과거가 아닌 지금의 공간을 제시."
- "지금의 공간을 보여주는 지도 SPOT."
- "초기 핵심 상권내 반복 사용과 데이터 밀도 확보."

## 문제 정의 (기존 서비스 한계)
- 기존 지도앱: 리뷰/사진은 과거 기록 → 지금 이 순간을 알 수 없음
- 인스타그램: 사후 업로드, 내 취향 사용자 존재 확인 불가
- 리뷰 플랫폼: 방문 후 후기, 실시간 정보 없음
- SPOT만이: 현재 혼잡도·사람 분포 확인 O / 실시간 현장 사진·영상 O / 내 취향 사용자 존재 확인 O / 정보 반영 시점: 실시간

## SPOT 핵심 가치
- "지금 이 순간, 내 취향의 사람이 어디에 있는지"
- 과거 기록이 아닌 현재 진행형 공간 정보
- MBTI 기반 취향 매칭 + 실시간 위치

## 타겟
- 20대 대학생 및 초기 직장인
- 카페, 술집 등 공간 방문 빈도 높음
- 분위기 기반 공간 탐색 니즈 높음

## 런치 전략
- 시작: 홍대, 성수 핵심 상권
- 인스타그램 15초 내외 문제해결형 숏폼 콘텐츠 30건
- 에브리타임 대학별 타깃 광고 30건
- 1일 100개 우산 총 15일간 홍대 입구역 6번 출구 배치
- 홍대, 성수 제휴 매장 8곳 QR 포스터 20장 배치
- 목표: 총 노출 100,000회 / 앱 설치 650건 / 초기 MAU 200명

## 성장 계획
- 2026 상반기: 초기 MAU 200명, 앱 설치 650건, 목표 매출 0.3억
